# 衛星データを用いた大規模異常検知システム

## プロジェクト概要
国立大学法人広島大学での研究プロジェクトにおいて、GCOM-C（しきさい）衛星データを用いた海洋漂流物（軽石・流れ藻）の自動検知システムを構築しました。

## 技術的特徴
- Big Data Handling: XarrayとDaskを活用し、数十テラバイトのマルチスペクトルデータを並列分散処理。
- AI推論: YOLOv5 を用いた高精度な対象物検知。
- 動態解析: 時系列解析により漂流物の流向・流速を自動算出し、船舶の安全航行に貢献。

## 成果
- 従来の手法では困難だった大規模データの処理パイプラインを構築し、研究サイクルを高速化。
- 突発的な海底火山噴火に伴う軽石流出イベントの早期発見をサポート。
